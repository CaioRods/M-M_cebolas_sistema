Passo a Passo para Instalação

No Windows

1

Clique no botão verde Baixar para Windows (.exe) acima.

2

Execute o arquivo M&M Cebolas Setup 0.1.2.exe baixado.

3

Abra o ícone M&M Cebolas criado na sua Área de Trabalho.

No macOS

1

Baixe a versão correspondente (Apple Silicon M1-M4 ou Intel).

2

Abra o arquivo .dmg e arraste o ícone M&M Cebolas para a pasta Aplicativos.

3

Se o Mac solicitar permissão no 1º acesso, permita em Ajustes do Sistema > Privacidade e Segurança.
